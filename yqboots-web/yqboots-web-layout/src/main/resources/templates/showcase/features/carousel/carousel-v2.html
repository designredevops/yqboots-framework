Responsive Carousel

OWL Carousel is touch enabled jQuery plugin that lets you create beautiful responsive carousel sliders. You can find more detailed information from the offecial plugin website http://owlgraphic.com/owlcarousel

Centered navigation with fixed width image

For centered navigation you should change class .owl-carousel-v1 to .owl-carousel-v2. You should add an additional class .owl-carousel-style-v1 to make fixed width image style.
