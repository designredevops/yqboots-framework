YQBoots Framework

YQBoots Framework is a production-ready framework for building modern applications. It is built on Spring IO Platform, and simplifying the initialization of a Spring-powered, production-grade applications and services with minimal effort by using Spring Boot.

Learn more »

The YQBoots Framework contains a fully responsive user interface, built by Bootstrap, providing an optimal viewing and interaction experience -- easy reading and navigation with a minimum of resizing, panning, and scrolling-across a wide range of devices (from desktop to mobile phones).

Features

The YQBoots Framework includes the most common parts of the enterprise applications.

  • User Interface -- A fully responsive, HTML5 + CSS3 user interface, no need to built different user interface for different devices.
  • Menu Management -- A maintainable menus displayed in the header.
  • Data Dictionary Management -- Manages the options in the html select elements
  • File Storage Service -- It allows you to upload and delete files in the configured folders under the specified root folder.
  • I18N Messages -- It supports multiple languages, and switch between them.
